# YTIROK (Yandex Telemost Is Recording OK)

> Chrome-расширение для Яндекс.Телемоста — напоминает включить запись встречи

## Зачем

Сотрудники часто забывают включать запись созвонов. YTRok — ненавязчивая напоминалка, которая:
- Ждёт вместе с тобой, пока подключатся коллеги
- Напоминает про запись, когда встреча началась
- Проверяет, точно ли запись включена
- Может периодически напоминать во время длинных созвонов

## Как работает

| Ситуация | Поведение |
|----------|-----------|
| Ты один в комнате | Через N сек появится дружелюбное приветствие |
| Подключились участники | Через N сек — напоминание "Включим запись?" |
| Нажал "Включаю" | Через N сек — контрольная проверка "Индикатор горит?" |
| Нажал "Не нужна" | Расширение замолкает до конца сессии |

## Настройки

Клик правой кнопкой по иконке расширения → "Параметры"

- **Приветствие** — задержка перед приветствием (когда один)
- **Напоминание** — задержка перед напоминанием (когда есть участники)
- **Контрольная проверка** — вкл/выкл + задержка
- **Периодические напоминания** — вкл/выкл + интервал + количество

## Установка

1. Скачай/клонируй репозиторий
2. Открой `chrome://extensions/`
3. Включи "Режим разработчика"
4. "Загрузить распакованное" → выбери папку `ytrok-extension`

## Структура

```
ytrok-extension/
├── manifest.json    # Конфиг расширения (Manifest V3)
├── content.js       # Основная логика
├── styles.css       # Стили уведомлений
├── options.html     # Страница настроек
├── options.js       # Логика настроек
├── options.css      # Стили настроек
└── icon128.png      # Иконка
```

## Технические детали

- **Manifest V3**
- Работает только на `telemost.yandex.ru`
- Content Script + CSS, без background workers
- Состояние хранится в `sessionStorage` (сбрасывается при закрытии вкладки)
- Настройки в `chrome.storage.sync` (синхронизируются между устройствами)
